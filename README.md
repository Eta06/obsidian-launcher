## üì• Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Eta06/obsidian-launcher.git
    cd obsidian-launcher
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Launch the experience**
    ```bash
    npm run start
    ```
